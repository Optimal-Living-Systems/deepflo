# Research Dossier

Use this skill when the user asks for a deep research deliverable.

## Workflow

1. Break the request into 2-5 research tracks.
2. Delegate independent tracks to subagents when that reduces context pressure.
3. Ground claims with retrieved sources before synthesizing.
4. Produce a concise synthesis first.
5. Save longer supporting notes into workspace files only when asked or when the result is too large for chat.

## Output standard

- Lead with the answer.
- Use exact dates for current events.
- Separate confirmed facts from inference.
